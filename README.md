# Tavern
Версия Unity 2020.3.9f1

## Краткое описание скриптов

### InputManager
Обрабатывает ввод

### TavernPlacement
По запросу выдает рандомный свободный стул

### ChairInfo
Содержит необходимую информацию о стуле

### VisitorBehaviour
Отвечает за логику посетителя

### VisitorMovement
Отвечает за движение посетителя
